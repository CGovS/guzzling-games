How to unlock buzzers →

Phone buzzers run on a tiny server on your computer — everyone joins over the same WiFi, free, no internet needed. It takes about 2 minutes the first time.

  1. Install Node.js 18+ (one-time) from nodejs.org — pick the “LTS” download.
  2. Download the game: go to github.com/CGovS/Liquid-Logic, click the green Code button → Download ZIP, then unzip it.
  3. Open Terminal (Mac) or Command Prompt (Windows) and run, one line at a time: 
    cd path/to/Liquid-Logic/server
npm install
npm start
    Tip: type cd  then drag the server folder into the window to fill the path.
  4. The window prints a HOST link and a QR code. Open the HOST link on your TV/laptop, flip on Live Phone Buzzers, and have players scan the QR (same WiFi) to buzz in from their phones.

Done playing? Close that window and the game goes back to keyboard buzzers. Full instructions are in the repo’s server/README.

How to Play

  • Setup: Enter team names, pick a theme and board mode.
  • Pick a Clue: Click any point value on the board.
  • Buzzing: Wait for the voice to finish reading! Buzz too early and you're frozen for 2 seconds once buzzers open.
    Keys: 1=6 (Teams 1–3)   ZB/ (Teams 4–6)
  • Answer Timer: Harder clues give more time — from 5 seconds at 200 up to 10 seconds at 1000. Wrong answer or timeout reopens the buzzer for other teams.
  • Moderated Mode: A host reads the clues aloud and marks who got it right — no voice or buzzers needed. Ideal for in-person play.
  • Scoring: Correct = points + assign sips. Wrong = lose points.
  • Sips: 200 = 1 sip · 400 = 2 sips · 600 = 3 sips · 800 = 4 sips · 1000 = 5 sips
  • Nobody gets it: the whole table drinks the clue's sip value.
  • Winning: Most points when all clues are used!

Drink responsibly. Water or soda recommended. This is a game of logic, not liver failure.

What is Liquid Logic?

Liquid Logic is the ultimate drinking trivia game — a fast-paced, Jeopardy-style showdown with over 6,000 questions across 50+ categories and 7 themed boards.

Teams buzz in, steal answers, assign sips, and compete for glory. Perfect for parties, game nights, and bars.

v6.0 Features

  • Live phone buzzers — run the local server and players buzz in from their phones on the same WiFi (scan a QR to join)
  • Works without a server too — falls back to keyboard buzzers and moderated play
  • Fully audited question set — corrected answers, retuned difficulty, no duplicate answers on a board
  • Scaled answer timer — more time on harder, higher-value clues
  • Real early-buzz penalty — jump the gun and you're frozen 2 seconds
  • Table-drinks rule when nobody gets the clue
  • Difficulty filter, live clue tracker, and end-game summary
